    • 情绪递进熟悉/好奇 -> 代入/停留 -> 被说中/心里一紧 -> 重新解释/松动 -> 被允许/想收藏或转发

    为什么能火 / 为什么有学习价值

    • 痛点够隐秘：想变好但很累不是用户每天公开说的话，但很容易在心里存在。
    • 入口够具体：先用标题、场景、物件或个人状态让用户进入，再讲抽象观点。
    • 情绪有转弯：从一件事转向自我理解，用户会等那个“原来如此”的瞬间。
    • 观点替用户说话：它把自责、羞耻或委屈换成一个更温柔的解释。
    • 互动理由明确：分享/赞 4.2%、评论/赞 5.6%，可谨慎推断有表达立场或讨论的动力；但评论文本缺失时不能伪造具体评论。
    这条的价值不是“失业时，家从来不是避风港”本身，而是用它承载了“想变好但很累”的情绪机制。

    红黄绿迁移清单

    绿色可学

    • 学习“生活小事 -> 深层情绪 -> 人生判断”的推进方式。
    • 先给具体场景/物件/处境，再讲观点。
    • 把自责转成重新理解，把痛点转成情绪许可。
    • 用短句和个人化表达降低说教感。

    黄色谨慎

    • 涉及出身、金钱、关系、创伤时要避免审判别人。
    • 长视频需要更强叙事控制，冷启动不要硬拖长。
    • 如果没有自己的细节，容易变成空泛鸡汤。
    • 过度脆弱暴露会给创作者造成心理和人设风险。

    红色不要复制

    • 不要复制对方原句、故事顺序和专属记忆。
    • 不要把对方具体人生经历改成我方第一人称真实经历。
    • 不要虚构创作者没有经历过的创伤、身份、收入、关系或成就。
    • 不要为了追热点把女性成长内容拍成炫耀、卖惨或导师训话。

    我方人设适配

    适配度高
    适合我方
    • 适合转成女性成长/情绪自救的口播，不需要站在观众上方教学。
    • 可以保留“我也是后来才明白”的朋友式语气。
    • 可以借“想变好但很累”引出反内耗和自我和解。
    不适合我方
    • 不适合照搬原作者的身份、经历、消费水平或关系背景。
    • 不适合变成强鸡血、强逆袭、强导师式口吻。
    要改什么
    • 更短句
    • 更像语音留言
    • 少用结论先行的大词
    • 保留一点疲惫但不绝望
    故事替换要求
    • 把具体经历改成创作者可确认的真实素材
    • 没有真实素材时使用低事实表达
    • 把题材服务于情绪自救，而不是复述原故事
